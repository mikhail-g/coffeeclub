# Personal Tasting Log

Tasting sessions are now tracked in Notion across two linked databases:

- **[Beans](https://www.notion.so/9a9ec97ce2934ca38b1a3b2f3f9378ad)** — one entry per unique coffee product: origin, process, price, roast profile
- **[Tries](https://www.notion.so/9d514be0aa5b4f1fad77973add43e294)** — one entry per tasting session: brew method, flavor notes, score, days off roast

Each Try links to a Bean. Each Bean links to a Roaster. Multiple tries per bean are supported.
